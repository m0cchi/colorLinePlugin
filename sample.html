Color Line Plugin

Color Line Pluginとは

最新鋭のデザインであなたのホームページを彩る事が出来ます.

導入方法は簡単!
Pluginをダウンロードし,パスの参照を出来る場所へ設置したら, 以下のコードをあなたのホームページに貼るだけ. 

    <link rel="stylesheet" type="text/css" href="./css/colorLine.css" />
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script type="text/javascript" src="./js/colorLine.js"></script>
    <style type="text/css">
    <!--
        html,body{
            width: 100%;
            height: 100%;
        }
    -->
    </style>
    <script type="text/javascript">
    <!--
        $(document).ready(function(){
            $(document.body).colorLine({lines:10,height:30,number:20});
        });
    -->
    </script>
